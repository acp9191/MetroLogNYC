# MetroLog NYC

An iOS app to track your NYC subway station visits.

## Features

- Track visited stations across all 470+ NYC subway stations
- View progress by line and borough
- Interactive map with color-coded station markers
- Filter stations by line, borough, or visited status
- Persistent data storage with SwiftData

## Requirements

- iOS 17.0+
- Xcode 15.0+

## Getting Started

1. Clone the repository
2. Open `MetroLogNYC.xcodeproj` in Xcode
3. Select your Development Team in Signing & Capabilities
4. Build and run on simulator or device

## Tech Stack

- SwiftUI
- SwiftData
- MapKit
